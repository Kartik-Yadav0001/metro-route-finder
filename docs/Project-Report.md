# Project Report

## Problem Statement

Design a metro navigation application that finds efficient routes and explains the algorithms used.

## Objectives

- Build a smart, responsive metro route finder.
- Demonstrate graph algorithms in a real-world context.
- Provide a clear admin workflow for network maintenance.
- Support documentation suitable for viva and portfolio review.

## Proposed Solution

Use an adjacency-list graph, a C++ route engine, JSON datasets, and a responsive React dashboard.

## Modules

- Frontend dashboard and route planner
- Backend graph engine and file manager
- Dataset storage in JSON
- Algorithm reference and documentation package

## Evaluation Highlights

- Weighted route optimization
- Search history
- Admin operations
- Modern UI/UX
