# Sample Inputs and Outputs

## Input

- Source: A
- Destination: G
- Mode: Shortest Distance

## Output

- Route: A → B → C → D → E → F → G
- Distance: 24 km
- Travel Time: 36 Minutes
- Fare: ₹45
- Stations: 7
- Interchanges: 2

## Edge Cases

- Same source and destination
- Disconnected station pair
- Invalid station code
- Empty history on first launch
