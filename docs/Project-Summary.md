# Metro Route Finder Project Summary

Metro Route Finder is a final-year, portfolio-ready metro navigation system built with a modern React frontend and a modular C++ DSA engine. It demonstrates real-world graph processing through weighted adjacency lists, route optimization, station search, and admin-driven graph updates.

## Core Features

- Station search with autocomplete
- Shortest, fastest, cheapest, and fewest-stops route finding
- Distance, time, fare, station count, and interchange calculation
- Interactive metro network visualization
- Search history tracking
- Admin CRUD for stations and tracks
- JSON dataset import/export and persistence

## Algorithms Used

- Graph adjacency list
- BFS for minimum stops
- DFS for connectivity checking
- Dijkstra for route optimization
- Floyd Warshall for all-pairs distance analysis
- Priority queue, hash map, queue, stack, greedy logic, sorting, and searching

## Technology Stack

- React.js, JavaScript, Tailwind CSS
- C++17
- JSON-based storage
- CMake build system

## Submission Value

This project is suitable for GitHub deployment, final-year evaluation, internship interviews, and viva presentation because it combines clean code, strong DSA coverage, a polished UI, and clear documentation.
